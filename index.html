Keeping the wonderful card game Smear alive with chickens!
Sponsored by Mouse Pup at mousepuptechnology@gmail.com
Currently in the testing phase, Smear.app is a four-player online traditional card game featuring a fantastic animated theme with barn mice and chickens. It utilizes Swift code, WebSocket protocols, and a great Sysync architecture. The code files are publicly available in the repository at https://github.com/kimmerandcasper/smearapp/tree/main
